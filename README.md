<h2>🚀 O Salto Tecnológico (v39.0)</h2>
<p>
  O <b>DescontoMEN-Bot</b> evoluiu de um monitor simples para um sistema de <b>Automação Robótica de Processos (RPA)</b>. Ele não apenas encontra ofertas, mas valida a integridade dos preços e automatiza a geração de links oficiais de afiliado via interface web (Browser Automation).
</p>

<h2>✨ Diferenciais de Engenharia & QA</h2>
<ul>
  <li><b>RPA Native Generator:</b> Automação via Selenium que acessa o Hub de Afiliados, gera e captura o link curto oficial <code>meli.la</code>, garantindo 100% de atribuição de comissão.</li>
  <li><b>Price Guard (Anti-Parcela):</b> Algoritmo de filtragem de contêineres HTML que ignora valores de parcelamento e foca exclusivamente no valor real do desconto.</li>
  <li><b>Limpeza de Milhar:</b> Parser de moeda brasileira que corrige distorções entre centenas e milhares, garantindo dados financeiros 100% precisos.</li>
  <li><b>Click1 Bypass:</b> Inteligência via Regex que converte links de anúncios patrocinados em links diretos de produtos para evitar rejeições no programa de afiliados.</li>
  <li><b>Strict Mode:</b> O bot prioriza a estética e a UX; se o encurtador oficial falhar, a oferta é descartada para manter o padrão de qualidade do grupo.</li>
</ul>

<h2>🛡️ Visão de QA (Quality Assurance)</h2>
<p>
  Como um projeto focado em <b>QA</b>, a resiliência é o pilar central. O bot gerencia sessões persistentes do Chrome (Cookies) para evitar logins repetitivos e utiliza esperas dinâmicas (Explicit Waits) para lidar com a latência do Mercado Livre.
</p>
